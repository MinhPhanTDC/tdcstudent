## MVP vs Full

**MVP (Phase 1-4)**: ~14 tuần
- Authentication hoạt động
- Admin quản lý cơ bản
- Student học và submit dự án
- Tracking tiến độ

**Full Features (Phase 1-8)**: ~24 tuần
- Tất cả tính năng MVP
- Chuyên ngành
- Lab Training
- Email system
- Polish & Production ready
